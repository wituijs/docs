# witUpdate 版本更新提示 %{#witUpdate}%

`witUpdate` 是一个基于 PWA (Progressive Web App) 封装的版本更新提示组件，用于自动检测应用更新并提供友好的更新提示界面。组件会在检测到新版本时自动显示更新对话框，并支持自动更新功能。

## 用法实列 %{#dome}%

```vue
<template>
  <div>
    <!-- 简单使用 -->
    <witUpdate />
  </div>
</template>

<script lang="ts" setup>
// 组件会自动检测更新，无需额外配置
</script>
```

## API

| 属性名 | 类型 | 默认值 | 描述 |
| :----- | :--- | :----- | :--- |
| 无 | - | - | 组件无需外部属性配置，自动处理更新逻辑 |

## 插槽

| 插槽名 | 描述 |
| :----- | :--- |
| 无 | 组件不提供外部插槽 |

## 事件

| 事件名 | 描述 | 参数 |
| :----- | :--- | :--- |
| 无 | 组件不对外暴露事件 | - |

:::tip
1. 组件基于 `virtual:pwa-register/vue` 实现 PWA 功能，需要确保项目已正确配置 PWA
2. 组件会在页面加载后自动检测更新，并在 3 秒后自动执行更新操作
3. 更新过程中会显示加载状态，更新完成后自动关闭对话框
4. 组件支持国际化，按钮文字会根据当前语言环境自动翻译
5. 组件样式已优化，提供了现代化的视觉效果，包括渐变色背景和图标
:::
